# 🔄 **Sensor Data Pipeline - Project Flow**

---

## **Complete Data Flow From Start to Finish**

```
START → Monitor Folder → File Detected → Validate → Transform → Aggregate → Store → END
         ↓                  ↓             ↓          ↓           ↓          ↓
      Every 5 secs      CSV file     Check      Standardize  Calculate   MySQL
                      in /data      quality     formats      stats      database
```

---
